# Admin Control Panel

This is a modern, professional, and responsive admin control panel built with Vite and React. It uses a Google Sheets Web App API as its backend to manage a list of codes.

## Features

- **Frontend**:
  - React with Vite as the build tool.
  - Professional, clean, and modern UI for managing codes.
  - Display a list of codes from a Google Sheet.
  - Buttons to **Add**, **Edit**, and **Delete** codes.
  - Responsive design with animations and transitions.

- **Backend**:
  - Google Sheets Web App API.
  - All data is stored and managed in a Google Sheet.

## How to Run Locally

### 1. Install Dependencies

First, you need to install all the necessary dependencies. Open your terminal in the project's root directory and run the following command:

```bash
npm install
```

### 2. Run the Frontend Development Server

You can start the frontend development server by running the following command:

```bash
npm run dev
```

The application will be available at `http://localhost:5173`. Any changes you make to the frontend code will be reflected in real-time.
